# 版本更新历史记录 (CHANGELOG)

本项目遵循语义化版本规范，用于记录“合同智能脱敏服务”各个版本的核心更新要点。

\---

## \[v1.2.0] - 2026-06-05

### 🚀 新特性与页面重构

* **物理多页面安全隔离**：

  * 新增独立的 `/mask`（文档脱敏）专属页面，彻底在 HTML 物理源码中删除了 Tabs 切换和规则配置相关 DOM，防止普通用户越权查看或篡改配置。
  * 新增独立的 `/restore`（文档还原）专属页面，物理隔离不必要的多余操作。
  * 保留 `/` 作为超级管理员三合一（脱敏、还原、配置）的总控制台。
* **自定义系统名称**：支持通过 [config.json](config.json) 里的 `"app\\\\\\\_name"` 参数自定义产品名称。
* **内置规则库扩容**：

  * 新增 **大写金额** 规则。
  * 新增 **数值金额** 规则。
  * 新增 **百分比** 规则。

### 🛠️ 稳定性与安全性优化

* **JS 底座防御性重构**：对 [app.js](app.js) 中的所有初始化与事件绑定模块进行了防御性 `null` 校验。即使在 DOM 被物理裁剪的独立功能页面中，也不会触发 `TypeError` 崩溃，实现了单套复杂逻辑跨页面 100% 稳定复用。
* **内置规则删除双向拦截**：将“大写金额”、“数值金额”与“百分比”加入前端和后端的内置标准黑名单。
* **版本号统一发布机制**：引入系统全局版本号并绑定至 FastAPI Swagger 元数据和控制台打印。

\---

## \[v1.1.0] - 2026-06-03

### 🎨 Ant-UI 视觉重构

* **Ant-Design 主题适配**：前端页面整体重塑为高质感 Ant-Design 扁平设计，完美适配亮/暗双色主题，并实现偏好一键切换及 `localStorage` 状态持久化。
* **Toast 位置优化**：将全局 Toast 通知框由顶部移动至屏幕底部居中，采用 `column-reverse` 倒序上浮动效，彻底规避遮挡顶部按钮的问题。

### 🌐 对外开放 OpenAPI 接口

* **增加OpenAPI**：新增无状态的一键自动脱敏 `/api/openapi/mask` 与文件流返回还原 `/api/openapi/restore` 接口，在 `/docs` Swagger 页面配备全中文化的详细参数说明，便于外部业务系统（如 OA、ERP）无缝对接。

### 🖥️ 跨平台启动与部署

* **跨平台启动器**：重写启动逻辑，支持 Windows 和 Linux 环境检测，首次启动自动执行 `pip install` 补全环境依赖。
* **Windows 命令行兼容性**：重新以纯 ASCII 编码写入 `run.bat`，解决 Windows 默认 GBK 代码页读取 UTF-8 导致 BOM 乱码。

\---

## \[v1.0.0] - 2026-05-28

### 🧬 核心脱敏算法

* **高保真格式保护**：实现段落 Run 级别跨文本切分重构算法，脱敏替换的同时 100% 保留 Word 原有段落、表格、字体、粗斜体、颜色等排版样式。
* **微软雅黑强制锁定**：使用 OxmlElement 底层操作对所有改动字符节点强锁 `ascii`、`hAnsi` 和 `eastAsia` 为 `'Microsoft YaHei'`，彻底消灭中西文混排字号忽大忽小与中文退化宋体 Bug。
* **双轨制安全还原**：

  * **本地一键静默还原**：利用内置属性 UUID 隐式配对 SQLite 字典。
  * **跨设备密钥还原**：导出备份 `.json` 密钥包以进行跨电脑在线恢复。
